# House Price Prediction using Linear Regression

## Description
This project predicts house prices using a real Kaggle dataset.
Multiple numerical features are used to train a Linear Regression model.

## Dataset
- Rows: 4600
- Target column: price

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Output
- R2 accuracy score
- Actual vs Predicted house price graph
